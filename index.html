🔄 Sync Strategy

เลือกได้ 2 แบบ — ดูรายละเอียดใน SYNC_DESIGN.md:

  • Push (Apps Script trigger): Sheet → onEdit → commit ผ่าน GitHub API → CF Pages auto-deploy
  • Pull (GitHub Action cron): daily 6am pull จาก Sheet → commit เปลี่ยนแปลง
  • Hybrid (แนะนำ): Push สำหรับ critical updates (events.js) + Pull สำหรับ static (suppliers.js)
